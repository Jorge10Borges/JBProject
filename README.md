# JBProject (Scaffold)

Scaffold inicial con Vite + Tailwind + PHP API.

## Requisitos
- Node.js 18+
- PHP (XAMPP) y Apache

## Instalación
```powershell
npm install
```

## Desarrollo
```powershell
npm run dev
```
Abrir: http://localhost:5173

## Build
```powershell
npm run build
npm run preview
```

## API
Servida por Apache/XAMPP en `/api`.
- `api/index.php` responde JSON de prueba.

Ajusta `CORS` y rutas según tu entorno.
